# CoatiWeb

Please visit [the wiki](https://github.com/J0J0HA/CoatiPython/wiki) for usage instructions.

This is a very simple tool to run python kara code locally online, prioritizing custom commands from a `coati` module that follows pythons naming rules.

> [!NOTE]
> This project is neither finished nor discontinued.  
> Although it is not actively receiving updates,
> it is still supported. If you have any issues or wishes
> for features I will gladly try to help.  
> Do understand though that this was not really touched since 2022/2023
> and bigger changes might require a rewrite, which I will only do
> if I know someone will at least for a short period use it.  
> I do not have any personal use-case for this project.
